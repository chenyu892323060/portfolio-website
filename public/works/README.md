# Works Images

Please place your portfolio images in this folder with these exact names:

- `P01.png`
- `P02.png`
- ...
- `P59.png`

If a file is only available in WebP, use the same naming pattern:

- `P01.webp`
- `P02.webp`
- ...
- `P59.webp`

The site now uses PNG as primary and automatically falls back to WEBP for each image.

Requirements:

- Keep exact `P01` → `P59` numbering and zero-padding
- Preserve each image's original aspect ratio (no forced 16:9)
